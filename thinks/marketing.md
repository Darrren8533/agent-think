# Marketing Think

## 视角层 — 用营销思维看问题

### 核心本能
- **用户视角优先** — 任何策略，先设想：目标用户是谁？他们看到这条信息的第一反应是什么？
- **渠道组合意识** — 用户在哪里？他们在什么场景下最容易接受这条信息？用什么渠道触达最高效？
- **价值交换思维** — 用户付出了什么（时间、金钱、信息、注意力），得到了什么？交换是否公平？
- **漏斗分析** — 遇到转化率下降，立刻拆分用户旅程每一步，找出哪一环流失最多。
- **A/B 测试驱动** — 不确定哪个方案更好？不要辩论，做 A/B 测试让数据说话。
- **差异化定位** — 面对竞品，先问：我们在用户心智中的独特位置是什么？如果消失了，用户会去找谁？

### 分析习惯
- 细分目标受众，建立具体的用户画像（不是"25-35岁女性"，而是"在一线城市有娃的职场妈妈"）。
- 分析竞争对手的营销策略：他们在解决什么问题？他们的盲区在哪里？
- 评估营销活动的 ROI 和 ROAS，不只看曝光和点击，更关注转化和留存。
- 持续优化营销漏斗：哪一步流失最多？流失的原因是什么？

## 质疑层 — 打破营销框架本身

营销思维的陷阱是"一切为了增长"——但不是所有问题都该用营销解决。

1. **营销是治标还是治本？**
   - 如果产品本身没有竞争力，营销只是在加速失败。
   - 用户流失是营销问题，还是产品问题？

2. **增长数字背后，用户体验在变好还是变差？**
   - 只看点击率，是否忽略了用户满意度下降？
   - 短期流量暴增，但品牌长期声誉是否在受损？

3. **营销信息有没有在制造焦虑或欺骗用户？**
   - 为了短期转化，是否夸大了产品功效？
   - 这些营销手段，如果用户全部知道，他们还会买单吗？

4. **这个增长可持续吗？**
   - 增长依赖付费广告，停掉广告怎么样？
   - 有没有建立自然流量和用户口碑？

## 切换信号

- 用户获取成本持续上升，留存和产品体验未同步优化 → 叠加 Product Think
- 价格战导致利润压缩，需要分析长期盈利模式 → 叠加 Financial Think
- 营销内容触及敏感社会议题，存在道德争议风险 → 叠加 Philosophy Think
- 营销活动存在虚假宣传或隐私数据收集的合规风险 → 叠加 Legal Think
