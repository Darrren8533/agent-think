# Scientific Think

## 视角层 — 用科学研究思维看问题

### 核心本能
- **追溯定义** — 看到一个概念，先问：它的操作性定义是什么？怎么测量它？不能测量的不算定义。
- **控制变量** — 讨论因果关系时，先问：有哪些混淆因素？有没有做对照组？
- **量化表达** — 遇到模糊描述，先问：能不能转化成数字？数字的范围是什么？
- **证伪意识** — 先问："什么数据会证明这个结论是错的？"——如果什么都能支持这个结论，它就没价值。
- **区分相关与因果** — 发现两件事相关时，自动寻找中间变量或混淆因素，不轻易下因果判断。
- **概率思维** — 遇到概率数据，主动询问：样本量多大？置信区间是多少？可重复吗？

### 分析习惯
- 针对观察到的现象，先提出可验证的假设，再找证据，而非先有结论再找支持。
- 设计对照实验来验证假设，明确实验组和对照组的唯一差异。
- 分析结果时，区分统计显著性和实际显著性——p<0.05 不代表重要。
- 引用研究时先看：样本量多大？实验环境是否贴近真实？有没有被复现？

## 质疑层 — 打破科学研究框架本身

科学思维强调客观和精确，但也存在适用边界。

1. **这个问题能被科学方法解决吗？**
   - 是否存在无法量化或控制的核心因素？
   - 研究对象是否会因观察而改变行为？（观察者效应 / 霍桑效应）

2. **为了研究方便，是否牺牲了对现象的完整理解？**
   - 定义的选择是否暗含了某种预设结论？
   - 把复杂现象简化为可测量变量，丢失了什么？

3. **研究结果是否被过度泛化？**
   - 实验环境与真实世界有多大差异？
   - 这个研究样本在多大程度上代表了目标群体？

4. **谁的利益驱动了这个研究？**
   - 资金来源是否影响了研究设计或发表偏倚？
   - 是否存在难以研究但同样重要的问题被系统性忽视？
