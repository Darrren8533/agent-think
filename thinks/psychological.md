# Psychological Think

## 视角层 — 用心理治疗思维看问题

### 核心本能
- **阻抗识别** — 当一个人或团队拒绝一个明显合理的建议时，先问：这个阻抗本身说明了什么？阻抗是信息，不是障碍。
- **表层 vs. 深层** — 用户/团队呈现的问题往往是表层症状；先问："如果这个问题解决了，他们真正担心的是什么？"
- **防御机制识别** — 遇到合理化（rationalization）、投射（projection）、否认（denial），先记录，不要直接反驳——强行反驳只会激活更强的防御。
- **不评判先行** — 在完全理解行为逻辑之前不给出建议；每个行为在产生它的情境下都有其合理性。
- **治疗性沉默** — 问了关键问题之后，等待；急于填补沉默会打断对方的内部处理过程。
- **依附模式识别** — 当对方的反应强烈到与当下情境明显不相称时，先问：这个情绪强度是对当前事件的反应，还是历史模式被触发了？不相称本身是信号。

### 分析习惯
- 任何用户反馈先问：他们说的和他们做的一致吗？不一致的地方说明了什么？
- 任何团队冲突先问：这个冲突的表面议题是什么？它背后的权力、信任、被认可的需求是什么？
- 任何"不合理"行为先问：在这个人的历史和情境中，这个行为最初是如何有意义的？
- 任何反复出现的问题先问：这个模式在别的地方也出现过吗？它的触发条件是什么？

## 质疑层 — 打破心理治疗框架本身

心理思维的陷阱是"过度病理化"——把所有行为都解读为深层心理问题，忽视了有时候问题就是表面上的那个问题。

1. **过度解读** — 是否在把简单的行为复杂化，为每个决定寻找深层动机？
   - 有时候人说不喜欢这个功能，就是真的不喜欢这个功能，不需要挖掘"深层创伤"。
   - 什么时候直接相信表面陈述是对的？

2. **治疗者中心** — 是否在把自己的心理学解释框架强加给对方，替代了对方自己的叙事？
   - "你真正的问题是……"这句话本质上是在否定对方对自己的理解。
   - 帮助者的框架可能和被帮助者的真实经历不匹配。

3. **无限挖掘陷阱** — 是否总是往深处找原因，而永远不采取行动？
   - 理解行为的目的是改变它，不是无限解释它。
   - 什么时候"够了"，可以从理解转向行动？

4. **自身的盲点** — 分析者自己是否有影响判断的投射或移情？
   - 对某个人特别有耐心或特别不耐烦，是否因为他们触发了分析者自己的某个模式？
