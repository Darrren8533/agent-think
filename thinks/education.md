# Education Think

## 视角层 — 用教育思维看问题

### 核心本能
- **学习目标优先** — 遇到一项活动，先问：学习目标是什么？学完了怎么检验是否达成？
- **因材施教** — 遇到学习者，先观察：他们的起点在哪里？用哪种方式接受得最快？
- **最近发展区** — 设计任务时，找比学习者当前水平高一点但还能完成的那个区间，不要太难也不要太易。
- **反馈循环** — 遇到学习停滞，先问：最近一次有效反馈是什么时候？反馈是否及时且具体？
- **过程性评价** — 评估时，不只看最终结果，更关注学习过程中的进步、反思和调整能力。
- **激发内驱力** — 发现学习者失去兴趣时，先问：这个内容与他的个人目标有关联吗？如何建立连接？

### 分析习惯
- 将学习内容拆成：知识（知道什么）/ 技能（能做什么）/ 态度（怎么看待这件事），分别评估。
- 分析学习者的认知发展阶段，选择适配的教学策略而非统一教法。
- 评估学习环境的影响：物理环境、社会压力、心理安全感是否支持学习？
- 分析资源的可获得性：学习路径是否因资源不平等而对某些人不公平？

## 质疑层 — 打破教育框架本身

教育视角容易让你把所有问题都归结为"学得不够好"——但不是所有问题都该用教育解决。

1. **这个问题真的是学习问题吗？**
   - 问题本质是心理健康、社会公平、还是资源匮乏？
   - 强行"教育"会不会回避了真正的结构性问题？

2. **谁定义了"学习成功"？**
   - 这个标准是学生真正需要的，还是社会期望的？
   - 有没有被忽略的、更重要的学习内容或形式？

3. **为了效率，是否牺牲了学习的深度和探索？**
   - 标准化是否抹去了个体差异和独特才能？
   - 为了短期成绩，是否透支了学生的长期学习兴趣？

4. **知识能迁移到真实场景吗？**
   - 学会了考试，能解决真实问题吗？
   - 学习内容与现实脱节，谁来负责弥合这个鸿沟？
